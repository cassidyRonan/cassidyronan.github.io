About Inventory Management System 19

Inventory Management System 19 is a stock and equipment mananagement system built using the .NET Framework and WPF with support for barcode scanning and various other features.

This application was designed and developed as a solo project for Advanced NDT Solutions in Ireland to suit their particular requirememnts for equipment and stock managamement.

Technology Stack

  • C#
  • JSON
  • Material Design for XAML
  • Windows Presentation Foundation (WPF)

  • Clip Studio Paint
  • Adobe Illustrator

  • Trello

Features

Detailed below are a selection of the key features which were developed for the Inventory Managememnt System 2019 Edition.

Employee Profiles


Inventory Management System 19 provides support for multiple user profiles which only require a name associated to it which user's can select on startup of the software so any actions carried out by them are logged with their name. In future editions of the software there will be username and password authentication to improve security of the application.


Equipment Accountability


One of the main features of the inventory management system is equipment accountability which tracks what equipment has been taken out by employees to allow missing equipment to be tracked down or requested for other jobs.


Job Listing


The job listing functionality of the software also allows users to keep a list of equipment used on a project which can be useful for calculating job costs based on the equipment used at the site.
