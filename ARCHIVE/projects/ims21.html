About Inventory Management System 21

Inventory Management System 21 is a stock and equipment mananagement system built using .NET Core 3.1 and a SQLite Database with support for multiple simultaneous users, barcode scanning and boasts several improvements over the original 2019 edition.

Technology Stack

  • C#
  • SQLite
  • Material Design for XAML
  • Windows Presentation Foundation (WPF)

  • Clip Studio Paint

  • Trello

Features

Detailed below are a selection of the key features which I am currently developing for the IMS 2021 Edition.

Material Redesign


The original version of the software in 2019 had a clunky early 2000's style aesthetic to it that resulted in the interface feeling clunky and unwieldly.

 

The software now has an updated interface following material design guidelines to result in a smoother and cleaner experience for the end user. This was accomplished with the help of the Material Design for XAML library.


SQLite Upgrade


The previous installment of the IMS software stored data via a JSON file and was only able to be accessed by a single user at once, leading to the possibility of item conflicts.

 

Now IMS 2021 has been rebuilt from the ground up utilising SQLite for the storage of data, allowing not only for portability to a new system but also the ability for multiple concurrent users to access the system.
