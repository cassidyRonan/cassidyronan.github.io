About AR Knight

AR Knight is an augmented-reality point and click action game as part of my fourth year project at the Institute of Technology Sligo.

The game is planned to be released on the Google Play Store later this year in Q4 2021.

Technology Stack

  • C#
  • AR Core
  • Unity AR Foundation
  • Unity Game Engine

  • Blender
  • Asset Forge
  • Clip Studio Paint

  • Trello
  • Microsoft Teams

Features

Detailed below are a selection of the key features which I developed for Project Crescent, these features are a combination of gameplay mechanics as well as behind the scene design systems.

Google Play Release


The game was originally slated to be released at the end of June 2021 onto the Google Play Store, however due to Google Play Store Policy changes, the project has been delayed until these issues can be resolved.


Scriptable Mission System


AR Knight contains a scriptable mission system which allows for simple and fast content generation to create new missions to push out to players at a moments notice. Part of this system includes the ability to add arena objects which act as interchangable platforms which the battling gameplay will be carried out on.
