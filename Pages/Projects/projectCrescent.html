Project Summary

Year: 2020
Project Status: Completed
Project Duration: 4 Months
Team Size: 2
Software Utilised: Unity 3D, Asset Forge, VRoid Studio, Clip Studio Paint, Trello, Microsoft Teams
Languages Utilised:   C#

Itch.io Link: Download The Game

Project Blog: View The Blog


Project Crescent is an action/ adventure open-world game built in the Unity game engine over the course of 4 months as part of a placement project for my BSc in Games Development at IT Sligo back in the summer of 2020. The project was developed by myself and a fellow classmate, the game was inspired by titles such as Final Fantasy, Breath of The Wild and Persona 5. The game was then released on the Itch.io marketplace at the end of the project's development period. My primary role in the project was lead programmer and world designer, however, over the course of the project I took on further development rolse such as model creation and combat designer.


Using my knowledge gained through my BSc in Games Development, I was responsible for core game mechanics design and implementation, some of the key features I worked on were the modular combat system, player controller and inventory system. As part of the project, I utilised Unity's Shader Graph to create a water shader reminiscent of the ocean from The Legend of Zelda: Wind Waker.
