Mech Commander VR


Project Summary

Year: 2019 - 2020
Project Status: Completed
Project Duration: 7 Months
Team Size: 3
Software Utilised: Unity 3D, Maya LT, Steam VR, Adobe Illustrator, Trello, Microsoft Teams
Languages Utilised:   C#

GitHub Repo: View on GitHub

Project Blog: View on Wordpress


Mech Commander VR is a virtual reality realtime strategy game or VR-TS as we called it internally in the team. It was developed as part of my third year project in the BSc in Computing in Games Development at IT Sligo alongside two fellow classmates. The project was developed as a proof of concept for the viability of a VR-TS where the player can control their units and use various abilities to turn the tide of the battle. My primary roles in the project was Project Lead and Lead Programmer, however, I also took on the roles of UX/UI Designer, Quality Assurance and Lead Artist.


Utilising my experience with game development and object oriented design, I was responsible for the design and implementation of both the user command center mechanics and base management mechanics. I also took on a support role in the project, filling any vacant roles in the development process due to my ability to quickly learn new technologies. This included the full implementation of virtual reality into the project utilising the Valve Index Controllers and Steam VR, 3D modelling and UI/UX design.
